¿Cómo obtener tu API Key gratuita?
1
Accedé a aistudio.google.com/apikey con tu cuenta de Google.
2
Hacé clic en Create API key y seleccioná un proyecto (o creá uno nuevo).
3
Copiá la key generada — comienza con AIza... — y pegala abajo.
4
La key gratuita incluye hasta 250 requests/día con Gemini 2.5 Flash. Suficiente para múltiples tesis.

La key se guarda localmente en tu navegador. No se envía a ningún servidor externo excepto directamente a la API de Google.
